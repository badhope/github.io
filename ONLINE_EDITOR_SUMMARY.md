## 🔧 技术实现

### 前端技术栈
- **HTML5** - 语义化结构
- **CSS3** - 现代化样式
- **Vanilla JavaScript** - 原生 JS 实现
- **GitHub API** - 文件操作
- **LocalStorage** - 配置和数据存储

### 核心技术

#### 1. 文件加载
```javascript
fetch(`https://raw.githubusercontent.com/${owner}/${repo}/${branch}/${filePath}`)
  .then(response => response.text())
  .then(content => { /* 加载到编辑器 */ });
```

#### 2. 文件提交
```javascript
// 1. 获取文件 SHA
fetch(`/repos/${owner}/${repo}/contents/${filePath}`)
  .then(res => res.json())
  .then(data => { /* 获取 SHA */ });

// 2. 更新文件
fetch(`/repos/${owner}/${repo}/contents/${filePath}`, {
  method: 'PUT',
  headers: { 'Authorization': `token ${token}` },
  body: JSON.stringify({
    message: '提交信息',
    content: base64 编码的内容，
    sha: data.sha
  })
});
```

#### 3. 文件树渲染
```javascript
// 递归渲染目录结构

```

#### 4. 本地保存

## 🐛 已知限制

### 技术限制
1. **文件大小**：建议编辑小于 1MB 的文件
2. **网络依赖**：需要稳定的网络连接
3. **浏览器性能**：大文件可能导致卡顿
4. **并发编辑**：可能发生版本冲突

### 功能限制
1. **语法高亮**：基础高亮，无智能提示
2. **代码格式化**：功能待实现
3. **版本历史**：需查看 GitHub 历史
4. **协作编辑**：不支持多人同时编辑
