#### 配置文件：`config/admin.config.js`
```javascript
const ADMIN_CONFIG = {
    username: 'admin',              // 唯一管理员用户名
    passwordHash: '...',            // SHA-256 加密的密码
    maxLoginAttempts: 5,            // 最大登录尝试次数
    lockoutDuration: 30,            // 账户锁定时间（分钟）
    sessionTimeout: 120             // 会话超时时间（分钟）
```

## 📋 配置步骤详解

### 步骤 1：修改用户名

编辑 `config/admin.config.js`:
```javascript
username: 'your_unique_username_2026'  // 修改为独特用户名
```

**建议**:
- ❌ 不要使用：`admin`、`administrator`、`root`
- ✅ 建议使用：`xiong_admin_2026`、`secure_user_888`

---

### 步骤 2：生成密码哈希

1. 打开 `utils/password-generator.html`
2. 输入强密码（建议格式）:
   ```
   示例：Xiong@2026Secure!Pass
   要求：
   - 至少 12 个字符
   - 包含大写字母（A-Z）
   - 包含小写字母（a-z）
   - 包含数字（0-9）
   - 包含特殊字符（!@#$%^&*）
   ```
3. 点击"生成哈希"
4. 复制生成的 64 位哈希值

---

### 步骤 3：更新配置

将生成的哈希值替换到 `config/admin.config.js`:
```javascript
passwordHash: '你的 64 位哈希值'
```

---

### 步骤 4：测试登录

1. 打开 `login-secure.html`
2. 输入配置的用户名和密码
3. 点击"安全登录"
4. 验证是否成功跳转到后台

## 🔧 常见问题

### Q1: 如何修改用户名？

**A**: 编辑 `config/admin.config.js`，修改 `username` 字段，然后重新部署。

---

### Q2: 如何修改密码？

**A**: 
1. 使用 `utils/password-generator.html` 生成新密码的哈希
2. 更新 `config/admin.config.js` 中的 `passwordHash`
3. 重新部署或等待自动部署

---

### Q3: 账户被锁定了怎么办？

**A**: 
方法 1（推荐）: 等待 30 分钟自动解锁

方法 2（开发环境）: 
```javascript
// 浏览器控制台执行
localStorage.removeItem('secure_auth_locked');
localStorage.removeItem('secure_auth_failed_attempts');
location.reload();
```

---

### Q4: 如何查看安全日志？

**A**: 
1. 登录到管理后台
2. 查看"最近安全日志"部分
3. 或点击"导出日志"下载完整日志

---

### Q5: 会话超时时间如何修改？

**A**: 
编辑 `config/admin.config.js`，修改 `sessionTimeout` 字段（单位：分钟）:
```javascript
sessionTimeout: 60  // 修改为 60 分钟
```
