🔑 GitHub Token 配置

配置 GitHub API 访问令牌以启用在线编辑功能

⚠️ 安全提示： Token 将仅存储在您的浏览器本地，不会上传到任何服务器。 请使用只针对此仓库的有限权限 Token。

如何创建 GitHub Token：

  1. 访问 GitHub Token 设置页面
  2. 点击 "Generate new token (classic)"
  3. 填写备注（例如：My Website Editor）
  4. 设置过期时间（建议 30-90 天）
  5. 勾选权限：repo（完整控制私有仓库）
  6. 点击 "Generate token"
  7. 复制生成的 token（只显示一次，请妥善保管）
💡 Token 格式通常以 "ghp_" 开头
您的 GitHub 用户名或组织名
包含网站文件的 GitHub 仓库名称
默认分支名称（通常是 main 或 master）
